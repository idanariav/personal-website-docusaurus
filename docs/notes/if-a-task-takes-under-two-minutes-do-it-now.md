# The 2 minute rule

## Notes

### Claim

if you can do this task in under 2 minutes, do it now.

### Explanation

This is meant as an alternative to prioritizing tasks [Group similar tasks to reduce switching costs](/notes/group-similar-tasks-to-reduce-switching-costs.md). Instead of spending time processing this task, there are simply cases where it is more efficient to directly do it rather than postpone it.  [Perfect is the enemy of good](/notes/perfect-is-the-enemy-of-good.md).
