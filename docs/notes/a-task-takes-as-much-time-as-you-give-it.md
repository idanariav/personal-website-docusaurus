# Parkinson's Law

## Notes

### Claim

A task will take as much time as you give it.

### Explanation

That means that we are more prone to [Procrastination is delaying intended actions despite preference to act](/notes/procrastination-is-delaying-intended-actions-despite-preference-to-act.md) if we have control on our schedule, meaning that the task is not with a hard deadline, or if the deadline is very far into the future.

But the reverse is true as well, if we give ourselves a very short deadline we can perhaps finish the task quickly, by using methods such as [the five minute rule](/notes/5-minute-rule.md).
